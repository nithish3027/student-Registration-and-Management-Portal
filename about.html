About Student Portal

A simple platform for student registration and management

What is Student Portal?

Student Portal is a simple web-based system created to register, view, and manage student information easily.

This project is developed using HTML, CSS, and JavaScript. It demonstrates basic web development concepts and simple interactive features.

Easy Registration

Students can register their information using a simple registration form.

Student Management

View and manage all registered student information in one place.

Simple Dashboard

View total students and basic student information from the dashboard.

Technologies Used

HTML

Used to create the structure of the website.

CSS

Used to design and style the website.

JavaScript

Used to add interactivity and functionality.
